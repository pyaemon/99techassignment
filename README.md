# 99techassignment

This repository contains solutions for the three assessment problems.

## Problem 1 - Sum to N

Implemented three approaches to calculate the sum of integers from 1 to n:

- Iterative solution
- Recursive solution
- Mathematical formula solution

## Problem 2 - Currency Swap UI

Built a responsive currency swap application using React and Vite.

Features:

- Token list display
- Real-time exchange rate calculation
- Token swap functionality
- Input validation
- Swap success confirmation
- Dark/Light mode support
- Responsive UI

## Problem 3 - Code Refactoring

Refactored the provided code to improve:

- Readability
- Maintainability
- Code structure
- Performance

## Run Locally

```bash
npm install
npm run dev
```
